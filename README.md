# Практична робота №7

### Завдання 7.1
Використайте popen(), щоб передати вивід команди rwho (команда UNIX) до more (команда UNIX) у програмі на C.

### Обгрунтування:
Реалізовано програму для передачі даних між двома незалежними процесами за допомогою функцій стандартної бібліотеки C.
Використано функцію popen() для запуску системної команди rwho. Програма відкрила потік для читання виводу цієї команди та перенаправила його у вільний дескриптор утиліти more.
В результаті отримано зручний посторінковий перегляд списку користувачів мережі, що демонструє принцип перенаправлення стандартних потоків введення-виведення.

### Завдання 7.2
Напишіть програму мовою C, яка імітує команду ls -l в UNIX — виводить список усіх файлів у поточному каталозі та перелічує права доступу тощо.
(Варіант вирішення, що просто виконує ls -l із вашої програми, — не підходить.)

### Обгрунтування:
Розроблено інструмент для зчитування метаданих файлів безпосередньо з індексних дескрипторів.
За допомогою викликів opendir() та readdir() програма сканує каталог, а для кожного об'єкта викликає stat(). З отриманої структури витягуються права доступу, кількість посилань, ID власника та розмір.
В результаті отримали повний аналог виводу ls -l, що відображає технічний стан файлової системи.

### Завдання 7.10
Напишіть програму мовою C для створення послідовності випадкових чисел з плаваючою комою у діапазонах:
(a) від 0.0 до 1.0
(b) від 0.0 до n, де n — будь-яке дійсне число з плаваючою точкою.
Початкове значення генератора випадкових чисел має бути встановлене так, щоб гарантувати унікальну послідовність.
- Примітка: використання прапорця -Wall під час компіляції є обов’язковим.

### Обгрунтування:
Створено генератор чисел з плаваючою комою у заданих діапазонах.
Використано rand() з нормуванням на RAND_MAX для отримання діапазону [0.0, 1.0]. Для унікальності при кожному запуску генератор ініціалізується через srand(time(NULL)).
Отримали послідовності унікальних чисел, необхідні для математичного моделювання.
